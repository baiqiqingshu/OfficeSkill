# PPT Slide Layout Examples

本文档提供各类幻灯片的具体布局参数示例，供 agent 在构建 officecli 提示词或 payload 时参考。

---

## 1. 封面页 (Cover Slide)

```
┌─────────────────────────────────────────────────────┐
│  ← 0.75in margin                                    │
│                                                     │
│                                                     │
│         [主标题 40-54pt, 居中或左对齐]                │
│         最多2-3行                                    │
│                                                     │
│         [副标题 20-28pt]                             │
│                                                     │
│                                                     │
│                                                     │
│         [演讲者 / 日期  14-16pt]                     │
│  0.5in margin →                                     │
└─────────────────────────────────────────────────────┘
```

**参数：**
- 标题区域：占幻灯片垂直中间 40%
- 上方留白：约 30%
- 下方留白：约 20%
- 左右边距：≥ 1.0 inch

---

## 2. 内容页 - 纯要点 (Bullet Content Slide)

```
┌─────────────────────────────────────────────────────┐
│  [幻灯片标题 28-36pt]                                │
│  ─────────────────────────────                      │
│                                                     │
│  • 要点一 (18-24pt, ≤ 10词)                         │
│                                                     │
│  • 要点二                                           │
│                                                     │
│  • 要点三                                           │
│                                                     │
│  • 要点四                                           │
│                                                     │
│                          [留白区域 ≥ 15%]            │
└─────────────────────────────────────────────────────┘
```

**参数：**
- 标题区域高度：幻灯片顶部 15-20%
- 标题与首个要点间距：≥ 0.4 inch
- 要点间行间距：≥ 0.2 inch
- 左侧文本起始位置：距左边缘 0.8-1.0 inch
- 右侧文本结束位置：距右边缘 ≥ 0.5 inch (若无图片)

---

## 3. 图文混排 (Image + Text Slide)

```
┌─────────────────────────────────────────────────────┐
│  [幻灯片标题 28-36pt]                                │
│  ─────────────────────────────                      │
│                                                     │
│  ┌──────────┐    • 要点一                           │
│  │          │    • 要点二                           │
│  │   图片    │    • 要点三                           │
│  │  (≤40%   │                                      │
│  │   宽度)   │                                      │
│  └──────────┘                                      │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**参数：**
- 图片区域：幻灯片宽度的 35-45%
- 图片与文字间距：≥ 0.4 inch
- 文字区域：幻灯片宽度的 45-55%
- 图片垂直居中对齐于内容区域

---

## 4. 双栏布局 (Two-Column Slide)

```
┌─────────────────────────────────────────────────────┐
│  [幻灯片标题 28-36pt]                                │
│  ─────────────────────────────                      │
│                                                     │
│  ┌─────────────┐  │  ┌─────────────┐               │
│  │  左栏内容    │  │  │  右栏内容    │               │
│  │  (≤45%宽)   │  │  │  (≤45%宽)   │               │
│  │             │  │  │             │               │
│  │  3-4条要点   │  │  │  3-4条要点   │               │
│  └─────────────┘  │  └─────────────┘               │
│                   ↑                                 │
│              间距≥0.5in                              │
└─────────────────────────────────────────────────────┘
```

**参数：**
- 每栏宽度：≤ 幻灯片宽度的 45%
- 栏间距：≥ 0.5 inch (幻灯片宽度的 10%)
- 两栏顶部对齐
- 每栏独立遵守内容密度规则 (≤ 25词/栏)

## 5. 数据图表页 (Chart Slide)

```
┌─────────────────────────────────────────────────────┐
│  [幻灯片标题 28-36pt]                                │
│                                                     │
│  ┌─────────────────────────────────────────┐        │
│  │                                         │        │
│  │         图表区域 (60-75% 面积)            │        │
│  │         标题 ≥ 20pt                      │        │
│  │         轴标签 ≥ 14pt                    │        │
│  │         数据系列 ≤ 7                     │        │
│  │                                         │        │
│  └─────────────────────────────────────────┘        │
│                                                     │
│  [数据来源 / 注释  12-14pt]                          │
└─────────────────────────────────────────────────────┘
```

**参数：**
- 图表占幻灯片面积：60-75%
- 图表与标题间距：≥ 0.3 inch
- 图表内标签字号：≥ 14pt
- 图例字号：≥ 12pt
- 数据来源注释：底部，12-14pt

## 6. 对比 / 表格页 (Comparison / Table Slide)

```
┌─────────────────────────────────────────────────────┐
│  [幻灯片标题 28-36pt]                                │
│                                                     │
│  ┌────────┬────────┬────────┬────────┐              │
│  │ 表头1  │ 表头2  │ 表头3  │ 表头4  │  ≥18pt Bold │
│  ├────────┼────────┼────────┼────────┤              │
│  │ 数据   │ 数据   │ 数据   │ 数据   │  ≥16pt      │
│  ├────────┼────────┼────────┼────────┤              │
│  │ 数据   │ 数据   │ 数据   │ 数据   │              │
│  └────────┴────────┴────────┴────────┘              │
│                                                     │
│  最多 5列 × 6行                                      │
└─────────────────────────────────────────────────────┘
```

**参数：**
- 表格最大尺寸：5 列 × 6 行
- 表头：Bold，≥ 18pt
- 单元格内容：≥ 16pt
- 单元格内边距：≥ 0.05 inch
- 如需更多数据，拆分为多页或使用附录

---

## 7. 引用 / 金句页 (Quote Slide)

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│                                                     │
│         "引用内容，简短有力，                         │
│          不超过两行。"                               │
│                        ── 引用来源                   │
│                                                     │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**参数：**
- 引用文字：24-36pt，居中
- 引用来源：16-20pt，右对齐
- 大量留白（≥ 50%）
- 引用内容不超过 30 词

---

## 8. 结束页 (Closing Slide)

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│                                                     │
│         [感谢语 / 结束语  36-44pt]                   │
│                                                     │
│         [联系方式 / 邮箱  14-16pt]                   │
│         [公司/组织名  14-16pt]                       │
│                                                     │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**参数：**
- 与封面页风格保持一致
- 信息极简（仅核心联系方式）
- 留白 ≥ 50%
